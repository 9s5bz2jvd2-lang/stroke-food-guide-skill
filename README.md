## 指南出处

- **全称**：《成人脑卒中食养指南（2026年版）》
- **发布单位**：国家卫生健康委办公厅
- **制定依据**：《健康中国行动（2019—2030年）》《国民营养计划（2017—2030年）》

## 功能特点

- **吞咽安全**：洼田饮水试验自测、食物质地分级、增稠剂使用指导（⭐脑卒中特色）
- **食养原则**：8条官方原则的详细解读与实操建议
- **营养管理**：能量/蛋白质/脂肪/碳水精确摄入量，DASH饮食模式
- **中医食养**：4种证型（痰热/痰瘀/阴虚/气虚）+ 4种体质 + 四季养生
- **地区食谱**：7大地区×4季节×3能量等级完整食谱
- **康复指导**：48小时即可开始活动，运动+吞咽+语言训练
- **冬季预警**：脑卒中冬季高发，保暖防复发要点
- **科普风格**：说人话、给具象量、指误区——精准但不端着

## 关键数据速查

| 项目 | 推荐量 | 通俗理解 |
|------|--------|---------|
| 食盐 | <5g/天 | 一个啤酒瓶盖 |
| 烹调油 | ≤25g/天 | 两汤匙封顶 |
| 饮水 | 1500-1700mL/天 | 约3瓶矿泉水 |
| 蛋白质（一般） | ≥1.0g/kg/天 | 60kg至少60g |
| 蛋白质（感染/压疮） | 1.2-1.5g/kg/天 | 加量补 |
| 脂肪供能比 | 20%-30% | 少油少肥肉 |
| 碳水供能比 | 50%-65% | 主食加粗粮 |
| 膳食纤维 | 25-30g/天 | 防便秘 |

**建设宗旨**：本Skill旨在普及脑卒中食养知识，响应全民营养周号召，落实《健康中国行动（2019—2030年）》和《国民营养计划（2017—2030年）》中关于发展传统食养服务、提升居民营养健康素养的要求，让权威食养指南走出文件柜，走进千家万户。
